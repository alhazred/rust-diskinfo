# rust-diskinfo
illumos diskinfo utility implemented in Rust

----

Example output:

```bash
TYPE    DISK                    VID      PID              SERIAL               SIZE
SCSI    c0t0d0                  SMC      SMC2108          005860522914ba4      930.39 GiB
SCSI    c0t1d0                  SMC      SMC2108          00a9aa522914ba4      930.39 GiB
```
